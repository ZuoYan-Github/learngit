4、删除链表中总乘积值为10的连续节点
     ->基本要求：链表结构每个元素中记录当前节点的值，和下个元素的地址。需要结果：经过删除连续节点的相乘结果不为10
     ->单链表算法思路：按照链表从头到尾进行遍历，当当前节点值与下个节点值乘积为10，获取下下个节点的地址与当前节点连接，删除下个节点，再次往下验证
	(按照注释内容，需要考虑链表中有无头节点与是否循环两种情况，是否双向对算法没影响）
     ->各种情况：链表种类：按照无头节点、单双向、是否循环分8中情况，这些情况由记录元素的链表结构定义，算法中需要分别处理
     ->衍生算法思路：
		->有无头节点：无头节点也是记录开始位置，区别在于开始位置是否读取数据进行判断
	              	->单双向：只考虑从头向尾排列，进行数据判断，两者没什么区别
		->是否循环：有头节点无所谓，无头节点需要与第一个元素进行数据判断
     ->C++设计：两个类，一个类是链表的类；另一个是对链表进行操作的类，两个函数：一是根据数组信息初始化链表，考虑各种情况，并输出链表；二是进行数据处理，完后输出新链表
